# Licoforice Anime Site

Статический сайт для просмотра аниме.

## Особенности
- Поиск аниме по названию
- Фильтрация по жанрам
- Адаптивная сетка

## Развертывание

### На GitHub Pages
1. Создайте репозиторий `licoforice-anime`
2. Загрузите все файлы проекта
3. В настройках репозитория выберите раздел "Pages"
4. В ветке `main` и папке `/root` активируйте GitHub Pages

### На Amvera
1. Создайте новый статический хостинг
2. Загрузите файлы через web-интерфейс или Git:
```bash
git remote add amvera https://username@amvera.ru/repository.git
git push amvera main
